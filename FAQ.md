# DeployWorkstation.ps1 - FAQ
## DeployWorkstation - Versions 1.0 -> 4.0 (10-6-2025)

### Frequent Questions & Answers:
- When you try to run the utility, you receive many Error -1978335217 messages on all application installations.
- Answer: So far this has always been related to internet connectivity/internet connection issues. Make sure the system is connected to the internet and you are able to surf/pass traffic without any issues. THEN re-run the utility.
